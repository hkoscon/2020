GitHub Actions: your free CI engine (and much more)

event 2020-06-12 (Day 1) ~ 16:35 - 17:05
explore Main Hall
  • comment English
  • network_check Intermediate

GitHub Actions is now the second most popular Build Platform on GitHub.

But it is also much more than than a CI system: it is a very powerful automation engine.

Let take a look at how we can do CI with Actions, and how we can automate operations on our GitHub projects.

Davide Benvegnu

Microsoft / Hong Kong

With over 16 years of experience in software development and IT, Davide is a DevOps Architect at Microsoft and GitHub, helping some of the biggest enterprise companies in Asia on their DevOps transformations.

Previously he's been Cloud Solution Architect for Microsoft, working with the Strategic Accounts.

Before that he was the CTO for DBTek, a Software House and IT consultancy company in Hong Kong. He's also been International Development Manager for Aruba, the biggest Italian web services provider. 

He has been awarded by Microsoft with the MVP (Most Valuable Professional) in ALM title in 2015, 2016 and 2017.
